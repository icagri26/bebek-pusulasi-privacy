# Bebek Pusulası — Gizlilik Politikası

Bu repo, **Bebek Pusulası** mobil uygulamasının gizlilik politikası sayfasını barındırır.

🔗 **Canlı sayfa:** https://ismailcagri26.github.io/bebek-pusulasi-privacy/

---

## Uygulama Hakkında

Bebek Pusulası, ebeveynlerin bebeklerinin büyüme sürecini takip etmelerine yardımcı olan bir mobil uygulamadır.

**Temel özellikler:**
- Büyüme takibi (boy, kilo, baş çevresi grafikleri)
- Aşı takvimi
- Gelişim kilometre taşları
- Bebek Bezi Sözlüğü (renk rehberi)
- Doktor Soru Defteri
- Çanta Kontrol Listesi
- Aile Senkronizasyonu (isteğe bağlı, oda koduyla)

---

## Gizlilik Özeti

- Kişisel kimlik bilgisi (ad, e-posta, telefon) **toplanmaz**
- Veriler öncelikle **cihazda** saklanır
- İsteğe bağlı senkronizasyon özelliği kullanılırsa veriler, kullanıcının oluşturduğu oda koduna bağlı olarak Firebase Firestore'da şifreli biçimde tutulur
- Reklam, analitik veya çerez **yoktur**
